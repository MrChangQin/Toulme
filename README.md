# 简历投递状态记录 Web 项目

此项目是一个简单的本地 Web 应用，用于记录简历投递状态。它包含 HTML、CSS 和 JavaScript 文件，可在浏览器中直接运行。

## 目录结构
- public/
  - index.html: 项目的主 HTML 文件，包含页面结构和表单元素。
- styles/
  - styles.css: 项目的 CSS 文件，用于设置页面样式。
- scripts/
  - script.js: 项目的 JavaScript 文件，实现表单提交和记录操作功能。
- res/
  - background.jpg: 页面的背景图片。

## 使用方法
1. 将这些文件保存到对应的目录下。
2. 在浏览器中打开 public/index.html 文件。
3. 在表单中输入公司名称、职位名称、投递日期和当前状态。
4. 点击“添加记录”按钮，将记录添加到表格中。
5. 若要删除记录，点击表格中的“删除”按钮。
